## Technical: `index.html` and running the app

`index.html` is the **HTML shell**: `lang="en-AU"`, favicon, viewport, `theme-color`, SEO meta, Google Fonts (**Inter**, **Sora**), `#root` boot placeholder, and `<script type="module" src="/src/main.tsx">`. Do **not** open it as `file://` — use the dev server.

```bash
npm install
npm run dev
```

Open the URL Vite prints (often `http://localhost:5173`).

| Location | Role |
|----------|------|
| `src/main.tsx` | React entry |
| `src/App.tsx` | Page sections order: Hero → Trust → Industries → Services → Portfolio → Process → Why Us → Testimonials → Pricing → FAQ → Contact |
| `src/components/` | UI sections |
| `index.html` | SEO, fonts, `#root`, noscript fallback |

To change **site-wide SEO** in the HTML shell, edit `<title>` and `<meta name="description">` in `index.html`.
